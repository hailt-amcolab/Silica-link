# Thư Mục Screenshots

Thư mục này chứa các ảnh màn hình để demo workflow.

## Cấu Trúc

Đặt các ảnh màn hình theo tên sau:

```
screenshots/
├── 01-shopify-order.png          # Màn hình Shopify - Khách hàng đặt hàng
├── 02-kintone-order-app.png      # Màn hình Kintone Orders App - Order mới
├── 03-inventory-check.png        # Màn hình Kintone - Kiểm tra tồn kho
├── 04-confirm-order.png          # Màn hình Kintone - Xác nhận order
├── 05-stock-updated.png          # Màn hình Kintone Stock App - Tồn kho cập nhật
├── 06-picking-task.png           # Màn hình Kintone Task App - Picking task
├── 07-warehouse-scan.png         # Màn hình Mobile App - Scan barcode
├── 08-shipping-label.png         # Màn hình Kintone - Shipping label
├── 09-customer-email.png         # Email gửi cho khách hàng
└── 10-complete-dashboard.png     # Dashboard Kintone - Tổng quan
```

## Cách Chụp Ảnh

### 1. Shopify Order Screen
- Chụp màn hình trang checkout của Shopify
- Hoặc trang order confirmation
- Format: PNG hoặc JPG
- Kích thước: 1920x1080 hoặc tương đương

### 2. Kintone Screenshots
- Chụp màn hình Kintone Orders App
- Chụp màn hình Kintone Stock App
- Chụp màn hình Kintone Task App
- Chụp màn hình Dashboard Kintone
- Format: PNG
- Kích thước: 1920x1080

### 3. Mobile App Screenshots
- Chụp màn hình mobile app (nếu có)
- Hoặc mockup mobile interface
- Format: PNG
- Kích thước: 375x667 (iPhone) hoặc tương đương

### 4. Email Screenshots
- Chụp email template
- Hoặc mockup email design
- Format: PNG
- Kích thước: 600x800 (email width)

## Tips

1. **Sử dụng placeholder nếu chưa có ảnh:**
   - File HTML sẽ tự động hiển thị placeholder
   - Có thể thêm ảnh sau

2. **Anonymize data:**
   - Che/mask thông tin nhạy cảm (tên khách hàng, địa chỉ, etc.)
   - Dùng fake data cho demo

3. **Consistent style:**
   - Dùng cùng browser/theme khi chụp
   - Cùng kích thước window
   - Cùng zoom level

4. **File naming:**
   - Dùng số thứ tự (01-, 02-, ...)
   - Tên mô tả ngắn gọn
   - Lowercase, dùng dấu gạch ngang

## Tools Để Chụp Ảnh

- **Mac:** Cmd+Shift+4 (select area), Cmd+Shift+3 (full screen)
- **Windows:** Snipping Tool, Win+Shift+S
- **Linux:** Screenshot tool, Print Screen
- **Browser:** Extensions như "Full Page Screen Capture"

## Mockup Tools (Nếu Chưa Có App)

Nếu chưa có app thật, có thể dùng mockup tools:
- **Figma** - Free, web-based
- **Balsamiq** - Wireframe tool
- **MockFlow** - UI mockup
- **Draw.io** - Simple diagrams
